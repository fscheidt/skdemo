# SvelteKit Template

## Clonar este template:

```bash
npx degit fscheidt/skdemo front
```

ou

## Criar um novo projeto

```bash
# cria um novo projeto chamado front
npx sv create front
```

## Instalar dependências

### devtools
```bash
npm install -D vite-plugin-devtools-json
```

## Iniciar a aplicação

```sh
npm run dev
```

iniciar e abrir no browser
```sh
npm run dev -- --open
```
